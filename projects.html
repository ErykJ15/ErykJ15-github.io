Pathfinding title

I used C# and the Monogame framework to create a program that uses the Dijkstra and A Star pathfinding algorithms as my final university project. Using JetBrains dotTrace for profiling and Visual Studio's performance and diagnostics centre, I carried out a technical analysis to compare their performance. Through the evaluation of memory and CPU utilisation on multiple weighted graphs with varying sizes and obstacles, I discovered that A Star consistently performed better than Dijkstra. A Star was a more effective option for real-time pathfinding applications because of its heuristic-based methodology, which allowed for quicker run times and reduced resource use.
